## Missing Information Prompts

### No reproduction steps
"What are the exact steps taken leading up to the issue?"

### No error details
"Can you share the full error message or API response?"

### No request payload
"Can you provide the request body or sample API call?"

### No environment context
"Which environment and account/tenant is this occurring in?"

### Intermittent issue
"When did this start, and is it consistent or intermittent?"

### Performance issue
"Is this affecting all requests or only specific endpoints/users?"
